# Unimplemented Language Features

This document lists language features that are defined in the AST but not yet implemented in the compiler/codegen.

## Control Flow Statements
- [ ] If/else statements
- [ ] While loops
- [ ] Do-while loops
- [ ] For loops
- [ ] Switch statements with case/default clauses

## Exception Handling
- [ ] Try/catch/finally blocks

## Expressions
- [ ] Binary expressions (arithmetic, comparison, logical, assignment operators)
- [ ] Unary expressions (prefix/postfix increment/decrement, logical not, unary plus/minus, bitwise not)
- [ ] Template literals
- [ ] Regular expressions

## Arrays and Objects
- [ ] Array literals
- [ ] Object literals
- [ ] Property access/initialization

## Functions
- [ ] Function expressions
- [ ] Arrow functions
- [ ] Method calls (function invocation)
- [ ] Async/await
- [ ] Default parameters
- [ ] Rest parameters

## Advanced Type System
- [ ] Union types
- [ ] Intersection types
- [ ] Generic types and type parameters
- [ ] Conditional types
- [ ] Infer types
- [ ] Template literal types
- [ ] Mapped types
- [ ] Type aliases

## Classes
- [ ] Class expressions
- [ ] Class getters/setters
- [ ] Abstract classes
- [ ] Class inheritance (extends)
- [ ] Interface implementation

## Interfaces
- [ ] Full interface declarations with methods, properties, index signatures

## Enums
- [ ] Enum declarations and members

## Destructuring
- [ ] Array destructuring
- [ ] Object destructuring

## Modules
- [ ] Import declarations (all variants)
- [ ] Export declarations (all variants)

## Parameter Handling
- [ ] Parameter lists
- [ ] Destructuring parameters

---

**Note:** The current compiler only supports basic variable declarations, block statements, class declarations with methods/properties, and simple expressions (literals, identifiers, member access, object construction via `new`).
